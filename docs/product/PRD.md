# PRD — 일감(ILGAM) MVP (2026-04-23)

## 문제 정의

한국 베이비부머 2차(1964~1974년생, 954만명)는 은퇴 직후 10~20년 경제활동을 희망하나:
- 기존 노인일자리 공공형은 월 30~60시간 고정 저단가
- 민간 스팟워크 1위 급구는 시니어 전담 UX 부재 (잠재 세그먼트로만 언급)
- 당근알바는 시니어 UX 분리 없음, B2B 대형사 연계 약함
- 60~74세 앱 이탈 60~70% (정보화진흥원 실증)

## 해결 가설

**한국형 시니어 스팟워크 플랫폼 = 타이미 매칭 엔진 + 3중 UX 방어선 + 공공 MOU 독점.**

- 타이미: 2024.7 도쿄증시 상장, 공모가 기준 시총 1,380억 엔(약 1조 3,400억 원), 가입자 1,000만+ (2024.12), 수수료 30% (KB경영연구소, 2025)
- 3중 UX: SMS 진입 + 오프라인 현장 + 동네지사
- 공공 MOU: 한국노인인력개발원·50플러스재단·구청 일자리센터
- 일감 수수료: 15~20% (타이미 30% 대비 보수)

## 시장 규모 근거

### 글로벌 긱이코노미
- 시장 규모: $5,567억(2024) → **$1조 8,470억(2032) · 3배+ 성장** (World Economic Forum, 2024)
- 디지털 플랫폼 근로자: 4,300만(2017) → **1억 6천만(2023) · 3.7배** (ILO, 2023)

### 일본 스폿워크 (벤치마크)
- 시장 규모: **496억 엔(2021) → 824억 엔(2023) · 연 30%+ 성장** (야노경제연구소)
- 4개 플랫폼 가입자 합계: **2,500만+** (2024.9)
- 스폿워커 수: **640만 명 (2023) · YoY +20.7%** (일본 후생노동성)
- 기업 효과 (타이미 2023 조사): **인사·급여·세무 관리비 18% 절감 · 전체 인건비 20% 절감**

### 한국 스폿워크 (출처: 통계청·고용노동부·KB경영연구소 2025)
- 주 15시간 미만 초단기 근로자: **153만(2020) → 182만(2024) · 역대 최고**
- 1년 미만 기간제 근로자: 354만(2020) → **388만(2024)**
- 급구 일자리 중개 건수: **12,480건(2017) → 335만 건(2024) · 300배+ · 연평균 +285.4%**
- 당근알바 공고 매칭 속도: **68%가 1시간 내 지원 마감**

### 시니어 수요 실증
- 일본 65세 이상: 3,625만 명(2024), 전체의 29.3% · 취업률 **25.2%** (Kyodo News, 2024)
- 일본 실버인재센터: "스폿워크 플랫폼이 고령자·경력단절 여성 노동시장 복귀에 실질적 기여" 공식 평가
- 미국 AARP 조사: **50세 이상 19%가 초단기 플랫폼으로 재취업** · 62%가 "전통 일자리보다 진입장벽 낮음"
- 한국 55~79세 장래근로 희망 비율: **69.4%**, 희망 월평균 근로소득 188만 원 (통계청 2024.8)

### 경쟁 지형 재확인
- 급구(니더): 2024년 중개 335만 건으로 시장 증명자 역할. 시니어 전담 프로덕트 **공식 런칭 선언 없음** (타이미 2025.7 릴리스에 "잠재 노동력"으로만 언급)
- 당근알바: 동네 기반 매칭 강점, **시니어 UX 분리 없음 · B2B 대형사 연계 약함**
- 알바몬·알바천국: 전통 이력서·면접 구조로 스폿워크와 별개 레이어

## North Star Metric

**월 완료 매칭 수** (see ADR-002)

## 타겟 (M1)

- **워커**: 서울 강서·송파·성북 거주 60~74세
- **구인자**: 수도권 중견 3PL + 베이커리·죽 프랜차이즈 본사
- **버티컬**: 물류·유통 + F&B

## 핵심 사용자 여정

### 워커 (시니어) — 첫 공고 지원까지 3탭
1. 알림톡 또는 SMS 링크 클릭 → 원터치 로그인 (매직링크)
2. 동네/요일/업종 태그 3개 선택 → 오늘 일감 리스트
3. "이 일 할래요" 원탭 → 확정 알림톡 (지원서·자기소개 없음)

### 구인자 — 공고 재게시 1탭
1. 어드민 로그인 (SSO 또는 매직링크)
2. 이전 공고 "다시 올리기" 클릭 → 시간·시급만 수정 → 게시
3. 지원 실시간 리스트 (Supabase Realtime) → 1탭 수락

### CX — 시니어 문의 처리
1. 알림톡 "문제 있어요" → 채널톡 카드 오픈
2. Claude intent 분류 → 0.85+ 자동답변 / 그 이하 상담원 이관 / 신고·급여는 즉시 사람
3. 디지털 실패 시 근처 동네지사 3곳 카드 + 예약 알림톡

## M1 기능 (3개월 내)

1. 원탭 지원 (ADR-002)
2. 카카오 알림톡 자동 리마인더 (ADR-004)
3. 동네지사 연동 온보딩 (ADR-002)
4. 구인자 반복공고 1클릭 재게시
5. PortOne 에스크로 정산

## 배제 기능

인앱 채팅 · 별점/후기 · AI 추천 피드 · 검색창

## 법·규제

- **통신판매중개업 신고** (자본금 요건 없음)
- **개인정보보호법 제29조·신용정보법 제19조** 기술적 보호조치
- **최저임금 2026년 기준 10,030원** 강제 검증 (DB 체크 제약)
- PortOne 에스크로 구조로 **전자금융거래법 PG업 등록 회피**
- 5년 회계·세무 보존 + 개인정보 파기의무 분리 (ADR-005)
