# PRD — 일감(ILGAM) MVP (2026-04-23)

## 문제 정의

한국 베이비부머 2차(1964~1974년생, 954만명) 및 70대 시니어(1955~1964년생)는 은퇴 직후 10~20년 경제활동을 희망하나:
- 기존 노인일자리 공공형은 월 30~60시간 고정 저단가
- 기존 민간 스팟워크 플랫폼은 시니어 전담 UX 부재 (잠재 세그먼트로만 언급)
- 기존 지역 알바 채널은 시니어 UX 분리 없음, B2B 대형사 연계 약함
- 60~74세 앱 이탈 60~70% (정보화진흥원 실증)

## 해결 가설

**한국형 시니어 스팟워크 플랫폼 = 매칭 엔진 + 3중 UX 방어선 + 공공 MOU 독점.**

- 수수료 정책: 15~20% (해외 레퍼런스 30% 대비 보수적)
- 3중 UX: SMS 진입 + 오프라인 현장 + 동네지사
- 공공 MOU: 한국노인인력개발원·50플러스재단·구청 일자리센터

## North Star Metric

**월 완료 매칭 수** (see ADR-002)

## 타겟 (M1)

- **워커**: 서울 강서·송파·성북 거주 60~79세 (70대 후반 포함 — 건강·근로 의지가 있는 모든 시니어)
- **구인자**: 수도권 중견 3PL + 베이커리·죽 프랜차이즈 본사
- **버티컬**: 물류·유통 + F&B

## 핵심 사용자 여정

### 워커 (시니어) — 첫 공고 지원까지 3탭
1. 알림톡 또는 SMS 링크 클릭 → 원터치 로그인 (매직링크)
2. 동네/요일/업종 태그 3개 선택 → 오늘 일감 리스트
3. "이 일 할래요" 원탭 → 확정 알림톡 (지원서·자기소개 없음)

### 구인자 — 공고 재게시 1탭
1. 어드민 로그인 (SSO 또는 매직링크)
2. 이전 공고 "다시 올리기" 클릭 → 시간·시급만 수정 → 게시
3. 지원 실시간 리스트 (Supabase Realtime) → 1탭 수락

### CX — 시니어 문의 처리
1. 알림톡 "문제 있어요" → 채널톡 카드 오픈
2. Claude intent 분류 → 0.85+ 자동답변 / 그 이하 상담원 이관 / 신고·급여는 즉시 사람
3. 디지털 실패 시 근처 동네지사 3곳 카드 + 예약 알림톡

## M1 기능 (3개월 내)

1. 원탭 지원 (ADR-002)
2. 카카오 알림톡 자동 리마인더 (ADR-004)
3. 동네지사 연동 온보딩 (ADR-002)
4. 구인자 반복공고 1클릭 재게시
5. PortOne 에스크로 정산

## 배제 기능

인앱 채팅 · 별점/후기 · AI 추천 피드 · 검색창

## 지표 (M3 말 목표)

| 지표 | 목표 | 근거 |
|---|---|---|
| 누적 워커 | 5,000명 | 창문 9~12개월 先手 락인 |
| 월 완료 매칭 | 1,500+ | 실패선 = 1,500 미만 |

## 법·규제

- **통신판매중개업 신고** (자본금 요건 없음)
- **개인정보보호법 제29조·신용정보법 제19조** 기술적 보호조치
- **최저임금 2026년 기준 10,030원** 강제 검증 (DB 체크 제약)
- PortOne 에스크로 구조로 **전자금융거래법 PG업 등록 회피**
- 5년 회계·세무 보존 + 개인정보 파기의무 분리 (ADR-005)
