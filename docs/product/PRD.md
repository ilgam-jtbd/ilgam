# PRD — 일감(ILGAM) MVP v3.1

> 일흔의 김씨는 오늘도 인력사무소 앞에 줄을 섰다.
> 새벽 다섯 시. 이름이 불리지 않으면 그냥 집으로 돌아간다.

인력사무소 디지털화 + 단계화 수수료 · 2026-04-28
연계 자료: `docs/ir/handoff.md` · `docs/decisions/ADR-002 v2.2` · `docs/decisions/ADR-004 v2.1` · `docs/decisions/ADR-011 v1.1`

## 김씨의 일주일

월요일은 공쳤다. 이름이 불리지 않았다.
화요일은 반나절. 시급은 미정이었다.
수요일에 받은 일당은 봉투에 담겨 왔다. 영수증은 없었다.
목요일 허리를 삐끗했지만 어디에도 적히지 않았다.
금요일 소장은 전화를 받지 않았다.

시니어 노동시장의 공백은 이렇게 한 사람의 한 주에 쌓인다.

## 앞서 간 사람들이 멈춘 자리에서

쑨은 명함을 모았고, 알바몬은 카테고리를 만들었고, 50플러스는 강의를 열었다. 그러나 김씨의 새벽은 그대로였다.

| 앞서 간 사람 | 시도 | 왜 안 됐나 | 우리가 다른 한 가지 |
|---|---|---|---|
| 쑨 (Shun) | 네이버 출신 50+ 매칭 앱 | 0%·광고 운영자본 소진 | 단계화 수수료 (M0~M5 → M6 10%) |
| 시니어즈 | 50플러스 직영 | 공공 예산 의존, SLA 부재 | KORDI MOU + 자동 매칭 |
| 알바몬·천국 시니어 탭 | 청년 채널에 카테고리 추가 | 청년 UI·신원검증 부재 | 시니어 UX (48dp/AAA/18pt) |
| 어르신일자리(공공) | 월 30시간 저단가 | 시급 한계·민간 단절 | 최저시급 10,030원 + 민간 풀 |
| 인력사무소(오프라인) | 새벽 줄서기·구두 매칭 | 디지털 전환 0%·사기 무방비 | 4-Tier QA + 디지털 신뢰 |

우리는 명함도 카테고리도 강의도 만들지 않는다. 다섯 시에 울리는 알림 하나, 끝나면 통장에 들어오는 일당, 허리가 아프면 누를 수 있는 버튼 하나. 손에 잡히는 것부터 바꾼다.

**한 줄**: 그들은 시니어를 '청년 모델의 하위 집합'으로 다뤘다. 우리는 시니어 시장의 구조 — 공공 신뢰·UX·사기 방어·수익 시점 — 를 처음부터 다시 설계한다.

## 문제 정의

한국 베이비부머 2차(1964~1974년생, 954만명) + 70대 시니어(1955~1964년생) + **욜드족(YOLD, 65~75세 대기업 출신)** 은 은퇴 직후 10~20년 경제활동을 희망하나:
- 기존 노인일자리 공공형은 월 30~60시간 고정 저단가
- 기존 민간 스팟워크 플랫폼은 시니어 전담 UX 부재 (잠재 세그먼트로만 언급)
- 기존 지역 알바 채널은 시니어 UX 분리 없음, B2B 대형사 연계 약함
- **사기·악성 공고 (성인·도박·MLM·고액일당 사기)가 시니어를 표적**으로 노출 — 신뢰 기반 채널 부재
- 60~74세 앱 이탈 60~70% (정보화진흥원 실증)
- **욜드족 경험 자산 사장**: 대기업 임원·CEO 출신이 단순 시급 노동에 갇힘 → 스타트업·중견기업 자문 매칭 시장 미개척

## 해결 가설 (v3.1 — 인력사무소 디지털화)

**인력사무소 디지털화 + 시니어 단독 점유 = Blue Ocean.** 5중 해자 결합:

1. **단계화 수수료**: Phase 0 양측 0% (M0~M5 검증) → **M6 법인 + 10% (인력사무소 오프라인 표준)** → Phase 2 + 광고/자문 멤버십
2. **시니어 전담 UX** (ADR-003)
3. **공공 MOU** (KORDI·50플러스·구청)
4. **컨텐츠 QA 시스템** (ADR-010 · 4-Tier)
5. **YOLD 자문 트랙** (consulting 카테고리, 50~64세 대기업·중견·스타트업 출신)

근거:
- 국내 인력사무소 ~3만곳 · 10% 수수료 표준 → 디지털 전환 공백 = 단독 점유 가능 시장
- 5개월 무료 검증 후 법인 설립과 함께 10% 도입 시 "인력사무소 표준 채택" 프레임 → 양쪽 거부감 최소화
- Seed ₩3억 + 정부과제 ₩5천만 → TIPS ₩10억 → Pre-A ₩10~15억 자금 트랙

## 시장 규모 근거

### 글로벌 긱이코노미
- 시장 규모: $5,567억(2024) → **$1조 8,470억(2032) · 3배+ 성장** (World Economic Forum, 2024)
- 디지털 플랫폼 근로자: 4,300만(2017) → **1억 6천만(2023) · 3.7배** (ILO, 2023)

### 일본 스폿워크 (해외 벤치마크)
- 시장 규모: **496억 엔(2021) → 824억 엔(2023) · 연 30%+ 성장** (야노경제연구소)
- 주요 플랫폼 가입자 합계: **2,500만+** (2024.9)
- 스폿워커 수: **640만 명 (2023) · YoY +20.7%** (일본 후생노동성)
- 기업 효과 조사: **인사·급여·세무 관리비 18% 절감 · 전체 인건비 20% 절감**

### 한국 스폿워크 (출처: 통계청·고용노동부·KB경영연구소 2025)
- 주 15시간 미만 초단기 근로자: **153만(2020) → 182만(2024) · 역대 최고**
- 1년 미만 기간제 근로자: 354만(2020) → **388만(2024)**
- 국내 스폿워크 중개 시장: **2017년 1만 건대 → 2024년 300만+ 건 · 연평균 +285.4%**

### 시니어 수요 실증
- 일본 65세 이상: 3,625만 명(2024), 전체의 29.3% · 취업률 **25.2%** (Kyodo News, 2024)
- 일본 실버인재센터: "스폿워크 플랫폼이 고령자·경력단절 여성 노동시장 복귀에 실질적 기여" 공식 평가
- 미국 AARP 조사: **50세 이상 19%가 초단기 플랫폼으로 재취업** · 62%가 "전통 일자리보다 진입장벽 낮음"
- 한국 55~79세 장래근로 희망 비율: **69.4%**, 희망 월평균 근로소득 188만 원 (통계청 2024.8)

### 시장 공백
- 국내 기존 스폿워크/지역 알바 플랫폼은 시니어 전담 프로덕트 **공식 런칭 선언 없음** (잠재 노동력으로만 언급)
- 동네 기반 매칭 강점 플랫폼은 시니어 UX 분리 없음 · B2B 대형사 연계 약함
- 전통 이력서·면접 구조의 알바 포털은 스폿워크와 별개 레이어로 시니어 미커버

## North Star Metric (v2 — 3-factor)

**MAVJ × MAU × TrustScore** (see ADR-002 v2)
- MAVJ = 월 활성 검증 공고 수 (qa_status='approved' + 1+ 지원)
- TrustScore = 1 − (신고/검증) − 자동반려율 보정. M3 베이스라인 0.7+

## 타겟 (M1)

- **워커**:
  - A 베이비부머 2차 (52~62세): 서울 강서·송파·성북, 일반 사무·생산·서비스
  - B 70대 시니어 (62~71세): 동일 지역, 사무·서비스·생활기술
  - **C 욜드족 (65~75세)**: 서울 전역 + 수도권, **대기업 임원·CEO·전문가 출신**
- **구인자**:
  - 수도권 중견 3PL · 베이커리·죽 프랜차이즈 본사 (트랙 A·B)
  - **시드~시리즈B 스타트업 + 매출 50~500억 중견기업** (트랙 C 자문)
- **버티컬**: 물류·유통 + F&B + **자문·멘토링 (consulting · 욜드족)**

## 핵심 사용자 여정

### 워커 (시니어) — 첫 공고 지원까지 3탭
1. 알림톡 또는 SMS 링크 클릭 → 원터치 로그인 (매직링크)
2. 동네/요일/업종 태그 3개 선택 → 오늘 일감 리스트
3. "이 일 할래요" 원탭 → 확정 알림톡 (지원서·자기소개 없음)

### 구인자 — 공고 재게시 1탭
1. 어드민 로그인 (SSO 또는 매직링크)
2. 이전 공고 "다시 올리기" 클릭 → 시간·시급만 수정 → 게시
3. 지원 실시간 리스트 (Supabase Realtime) → 1탭 수락

### CX — 시니어 문의 처리
1. 알림톡 "문제 있어요" → 채널톡 카드 오픈
2. Claude intent 분류 → 0.85+ 자동답변 / 그 이하 상담원 이관 / 신고·급여는 즉시 사람
3. 디지털 실패 시 근처 동네지사 3곳 카드 + 예약 알림톡

## M1 기능 (3개월 내, v2)

1. 원탭 지원 (ADR-002)
2. 카카오 알림톡 자동 리마인더 (ADR-004)
3. 동네지사 연동 온보딩 (ADR-002)
4. 구인자 반복공고 1클릭 재게시
5. **컨텐츠 QA 큐 (운영자)** — ADR-010, 분당 5~10건, P0 (이전 #5 PortOne 정산 폐기)

## 의도적 배제 5건

- **수수료 시스템** (Phase 1) — 양측 0%. PortOne 정산 흐름 폐기 (직거래).
- **인앱 채팅** — 채널톡 단일화
- **별점·후기** — 초기 부정 편향 폭주 위험
- **매칭 자동 LLM** — 시급/거리/시간 결정형 (Claude는 QA·CX만)
- **검색창** — 시니어 인지부하 (위치·요일·업종 3 태그 + 동네지사 직원 상담)

## 법·규제 (v2)

- **통신판매중개업 신고** (자본금 요건 없음). 직거래 모델로 자금 흐름 미개입 = 전자금융거래법 우려 자체 소멸.
- **개인정보보호법 제29조·신용정보법 제19조** 기술적 보호조치
- **최저임금 2026년 기준 10,030원** 강제 검증 (DB 체크 제약)
- 5년 회계·세무 보존 + 개인정보 파기의무 분리 (ADR-005)
- **광고 매출 도래 시(M18+)**: 광고 수익 부가세·소득세 처리, 광고주 세금계산서 별도 워크플로 (전략팀장 사전 검토 — `docs/architecture/security.md` "검토 중" 마커)
