# 일감(ILGAM) 전략 피벗 결정서 v3.0

**결정 일자**: 2026-04-24
**결정자**: 김연재 (Product Owner)
**문서 버전**: v3.0 (이전 v1.0 = 2026-04-23 ADR 8건 기반 수수료 모델)
**상태**: Approved · 즉시 시행

## 1. 결정 요약

일감(ILGAM)의 사업 모델을 **수수료 기반 양면 매칭 플랫폼**에서 **무료 시니어 잡 디스커버리 플랫폼 + 컨텐츠 QA + Phase 2 광고 수익화**로 전환한다. 단일 앱 구조로 구인자·구직자가 동일 인터페이스를 공유하며(당근알바 모델), 운영자 백오피스만 분리한다.

본 결정은 2026-04-23 작성된 ADR-002, ADR-004, ADR-005 일부의 핵심 가설을 폐기하고, 6개월~24개월 시계열 비즈니스 운영 방식을 재정의한다.

## 3. 결정 근거

### 3.1 양면시장 점화 실패 리스크 회피

스팟워크 양면시장에서 수수료 부과는 양면 모두에 진입 마찰을 일으킨다. M1~M3 워커 5,000명·구인자 200사 동시 확보가 핵심 가설이었으나, 수수료 정책이 양측 가입 의사결정을 지연시키는 구조적 리스크가 확인되었다. 무료 모델은 진입 마찰을 0으로 만들어 양면 점화 속도를 가속한다.

### 3.2 당근알바 모델 검증

당근알바는 누적 가입자 3,800만 · MAU 2,100만 · 단기알바 비중 22% · 공고 68%가 1시간 내 마감(KB경영연구소 2025.06)이라는 결과를 무료 + 광고 모델로 달성했다. 시니어 특화·공공 연계·컨텐츠 QA 깊이가 부재한 상태에서도 시장 자체는 무료 모델의 작동을 입증했다. 일감은 동일 모델 위에 4중 차별화를 얹는 구조로 시니어 시장에서 경쟁 우위를 확보한다.

### 3.3 컨텐츠 QA의 전략적 가치

당근알바·시장 전반의 약점은 **공고 신뢰 부족**이다. 가짜 공고·MLM·다단계·고액일당사기·성인 접대 위장 공고가 시니어를 직접 타겟한다. 일감의 4-tier QA 시스템(자동 키워드·Claude API·운영자 검토·외부 신고)은 시니어 신뢰 자산을 축적해 무료 + 광고 모델의 LTV를 결정짓는 핵심 해자가 된다.

### 3.4 광고 수익화의 시점 정합성

광고 매출은 MAU 5만+·검증 공고 월 5천+ 시점에서 의미있는 매출을 발생시킨다(당근알바·Indeed 보수적 추정). M18 시점 도달이 합리적이며, 그 사이 24개월 러웨이를 Seed 3~5억 + Pre-A 10~15억으로 확보한다.

## 4. 영향 분석 (8개 영역)

### 4.1 제품 (Product)
- 단일 앱 구조: apps/admin-web의 구인자 영역과 apps/worker-app을 통합 모바일 앱으로 흡수. 운영자 백오피스는 별도 웹 인터페이스 유지.
- NSM 재정의: "월 완료 매칭 수" → "월 활성 검증 공고 수 × MAU × 신뢰 지수" 3-factor.
- 정산·결제 흐름 단순화: PortOne 에스크로 폐기, 광고 결제용 보조 인프라로 격하.

### 4.2 기술 (Engineering)
- DB 스키마 재설계: `payments` → `ad_payments` 재명명 또는 폐기, `platform_fees_daily` 폐기.
- jobs 테이블에 qa_status·qa_reason·qa_classifier 컬럼 추가.
- Edge Function `qa-classifier` 신설 (Claude API 연동).
- 운영자 백오피스 우선순위 재정렬: QA 큐가 P0.
- 광고 플랫폼 인프라 dormant 준비 (M18 트리거 시 활성).

### 4.3 재무 (Finance)
- BEP 재산정: M10~M11 → **M22~M28 (시나리오별)**.
- 누적 적자 운영: 약 1.5~4억 (24개월 러웨이).
- Seed 자금 증액: 1~2억 → **3~5억**.
- Pre-A: 5~10억 → **10~15억**.
- 광고 매출 추정: 보수 월 1,175만 / 기본 월 3,225만 / 낙관 월 7,000만+ (Phase 2).

### 4.4 법적 (Legal)
- 통신판매중개업 신고 → **검토 필요**: 수수료 0%면 단순 정보매개 모델 해당 가능성. 전자상거래법 적용 범위 재검토.
- PortOne PG업 등록 회피 논거 → 무관(에스크로 미사용).
- 광고 매출 발생 시점에 광고 표시 의무·소비자 보호 규정 적용.
- 임금체불 1차 보전 책임 재검토 필요(KORDI MOU 11번 조항).

### 4.5 인사 (HR)
- 컨텐츠 QA 운영자 1명 채용 (M3 이후, 월 200~250만원).
- M12 이후 광고 영업 1명 추가 (M18 매출 시작 대비).
- 김연재·CPO·전략팀장 직함 정책 변동 없음.

### 4.6 파트너십 (Partnership)
- 한국노인인력개발원 MOU 제안서 v2 갱신: 수수료 5%p 할인 폐기 → 공공 추천 워커 우선 노출 슬롯으로 교체.
- 당근알바 비교 전략 추가 (직접 경쟁자 격상).
- 전략 자본 확보 명분 변경: BEP 단축 → 24~30개월 러웨이.

### 4.7 마케팅 (Marketing)
- "수수료 보수적" 메시지 폐기 → "양측 무료 + 검증된 시니어 일자리" 신규 메시지.
- 한국노인인력개발원·50플러스재단·동네지사 3축 채널 유지.
- 시니어 신뢰 강조: "성인·도박·MLM·사기 공고 차단" 명시.

### 4.8 QA (Content & Service)
- 컨텐츠 QA 4-tier 시스템 신규 운영 (자동·Claude API·운영자·신고).
- Claude API 비용 추정: 월 1만건 × $0.005 ≈ 7만원.
- 운영자 분당 5~10건 처리 목표 UX.
- 신뢰 지수 산정 로직 신설 (NSM 3-factor 중 하나).

## 5. 폐기되는 가설 3건

- **H1**: "수수료 15~20%로 BEP M10~M11 도달 가능" → 무료 모델로 BEP M22~M28로 후행
- **H2**: "수수료 매출이 양면시장 점화 동력" → 무료가 양면 점화 정공법 (당근알바 검증)
- **H3**: "PortOne 에스크로가 정산 핵심 인프라" → 광고 결제용 보조 인프라로 격하

## 6. 신규 가설 3건

- **H4**: 양측 무료 + 시니어 UX + 공공 MOU = 6개월 내 워커 5,000명 + 검증 공고 5,000건 도달
- **H5**: 컨텐츠 QA 4-tier 깊이가 당근알바 대비 핵심 차별화이며, 시니어 신뢰가 LTV·바이럴을 결정한다
- **H6**: M18 시점 MAU 5만+ · 검증 공고 월 5천+ 도달 시 광고주 모집 가능 (광고 매출 인플렉션)
