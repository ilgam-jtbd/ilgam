## R1. 광고 도달 실패

### 시나리오
M18 시점에 MAU 5만 + 검증 공고 월 5천 도달 가설 미달성. M18 도달했으나 MAU 1.5~3만, 검증 공고 월 1.5~3천 수준에 머무름.

### 발생 확률: 중 (40%)
당근알바 모델은 동네 지역사회 침투 + 카카오톡 자연 바이럴 결합으로 빠른 MAU 확장이 가능했다. 일감은 시니어 타겟이라는 인구통계 제약 + 디지털 격차 60~70%(정보화진흥원)이라는 구조적 마찰을 안고 있다. 단순 비례 적용 시 동등 시점 MAU의 30~60% 수준에 도달할 가능성이 충분히 있다.

### 영향: 최대
- 광고 매출 인플렉션 시점이 M22~M24로 후행
- BEP 도달 시점 M28~M32로 연기
- Pre-A 라운드 자금 규모 + 1.5~3억 추가 필요
- 회사 가치 평가 하향 가능성

### 완화책
1. **공공 MOU 채널 강화**: 한국노인인력개발원 산하 1,500개 수행기관 + 50플러스재단 채널을 통한 안정적 워커 유입. 단순 디지털 마케팅 의존도 축소.
2. **동네지사 거점 확장**: M9까지 강서·송파·성북 외 2~3개 구 추가. 디지털 이탈자 흡수.
3. **B2B 영업 선행**: M6부터 중견 3PL·F&B 프랜차이즈 본사 직접 영업으로 검증 공고 베이스 확보. 광고주 모집 못 해도 B2B Premium 매출 조기 발생 가능.
4. **MAU 측정 기준 강화**: 월 1회 접속 → 월 2회 + 1건 이상 검색 → 월 3회 + 검색·지원 활동 (피벗 측정)

## R2. 컨텐츠 QA 외주 인건비 폭증

### 시나리오
공고 등록량이 월 1만건에서 5만건으로 5배 증가. 자동 분류 + Claude API + 운영자 검토 흐름에서 운영자 처리량이 한계에 도달, 운영자 1명 → 3명 채용 필요.

### 발생 확률: 중 (35%)
M9~M12 시점 정상적 성장 패턴이며, 자동화 비율을 70% 이상으로 유지하지 못할 경우 인건비 폭증 가능.

### 영향: 중 (월 200~250만 → 월 600~750만)
- 24개월 누적 추가 비용 약 1억
- Pre-A 자금 규모 +0.5~1억

### 완화책
1. **자동 분류 정확도 개선**: Claude API 프롬프트 정기 갱신, few-shot 학습 데이터 축적. confidence 0.85+ 자동 처리 비율 80%+ 목표.
2. **블랙리스트 키워드·URL 패턴 자동 학습**: 운영자 반려 사유 분석으로 자동 필터 강화.
3. **외부 신고 인센티브**: 신고 적중 시 워커에게 활동 점수 부여 (광고 슬롯·동네지사 우선권 등). 사용자 참여형 모더레이션.
4. **AI 어시스턴트 운영자 도구**: 운영자가 단일 화면에서 결정 1초 내 가능하도록 UX 최적화. 분당 10건 → 15건 목표.

## R3. 무료 가입자 진성도 저하

### 시나리오
무료 모델로 가입 마찰이 0이 되면서 진성 시니어 워커 외에 일회성 가입자·중복 가입·비활성 가입자가 다수 유입. NSM의 신뢰 지수가 하락.

### 발생 확률: 중하 (30%)
SMS 본인인증 + PASS·NICE 본인확인을 거치므로 중복 가입은 차단. 다만 비활성 가입자(가입만 하고 사용 안 함) 비율은 무료 모델 특성상 50% 이상 가능.

### 영향: 중
- MAU 통계 부풀려져 광고주 신뢰 저하 (Phase 2 영향)
- 매칭 풀의 실효성 저하 (구인자 만족도 하락)

### 완화책
1. **활성 워커 정의 강화**: "최근 30일 내 1회 이상 검색 또는 지원 + 알림톡 수신 동의"로 한정.
2. **비활성 워커 자동 휴면**: 90일 미접속 시 휴면 처리, 광고주 노출 제외.
3. **워커 신뢰 점수 도입**: 출근률·평점·근속을 종합하여 점수화, 구인자에게 노출.
4. **MAU 외 보조 지표**: WAU·DAU·검증 공고 수·매칭 성사 수를 함께 공개하여 투명성 확보.

## R4. 광고주 모집 실패 (시니어 광고시장 협소)

### 시나리오
M18 시점 MAU 5만 도달했으나 시니어 타겟 광고주 풀이 협소하여 광고 단가가 보수 추정의 50% 이하로 형성. 또는 광고 슬롯 판매 자체가 부진.

### 발생 확률: 중상 (45%)
시니어 타겟 광고시장은 2024년 기준 약 3~5조원 규모로 추정되나 디지털 광고 비중은 낮음. 일감 광고 슬롯은 채용 광고에 한정되어 더 협소.

### 영향: 최대
- Phase 2 매출 추정의 50% 미달 시 BEP M30+ 후행
- 자본 추가 조달 또는 사업 모델 재검토 (B2B Premium 비중 강화) 필요

### 완화책
1. **B2B Premium 우선**: 광고 매출의 50% 이상을 B2B Premium(월정액)으로 확보. CPC 의존도 축소.
2. **광고주 후보 사전 확보**: M12까지 한국노인인력개발원·50플러스재단·구청 일자리센터·중견 3PL·F&B 프랜차이즈와 사전 광고 의향서(MOU 형태) 확보.
3. **익명 데이터 사업 강화**: 시니어 노동시장 데이터를 지자체·연구기관·정책 연구원 대상 정기 구독화.
4. **광고 슬롯 다양화**: 채용 광고 외 시니어 친화 상품·서비스 (건강·보험·교육) 광고 영역 확대 검토 (M24 이후).

## R5. 법적 지위 재검토 부정적 결과

### 시나리오
법무 자문 결과 무료 매개라도 통신판매중개업 신고 의무 발생, 또는 광고 매출 발생 시점에 추가 규제(광고 표시 의무·소비자 보호) 적용. 컨텐츠 QA 의무가 법적으로 강화되어 운영 부담 증가.

### 발생 확률: 하 (15%)
한국 전자상거래법은 매개 행위 자체에 대한 규제이므로 수수료 유무와 무관할 수 있다. 광고 매출 시점에는 표시·광고의 공정화에 관한 법률 적용 분명.

### 영향: 최대
- 사업 모델 재구조 필요
- M3 이전에 결과가 나오면 영향 최소화 가능

### 완화책 (P0 — 가장 빠른 확인 필요)
1. **외부 법률 자문 즉시 의뢰**: 30일 내 통신판매중개업 신고 여부, EU 플랫폼 노동 지침의 한국 적용 가능성, 광고 표시 의무 검토.
2. **개인정보보호법·신용정보법 충족 설계 유지**: 피벗 후에도 ADR-005 PII 분리 정책 변경 없음.
3. **선제적 컨텐츠 QA 운영 매뉴얼 문서화**: 법적 의무 강화 시 즉시 적용 가능한 4-tier 시스템 운영 절차서 사전 작성.
4. **법무 자문 결과 따른 ADR 갱신**: 결과 도출 시 docs/decisions/ADR-009·010에 반영.

## R6. 욜드 자문 풀 모집 실패

### 시나리오
M12 시점에 자문 가능 욜드 5,000명 풀 미달성. 1,000~3,000명 수준에 정체. 5번째 차별화 축이 약화되고 자문 트랙 매출(Phase 2) 기여도 미달.

### 발생 확률: 중 (40%)
대기업 임원 출신 풀(60만명)은 충분하나, 디지털 플랫폼을 통한 자문 활동에 대한 거부감·체면·기존 인맥 네트워크 의존도가 진입 마찰로 작용한다.

### 영향: 중상
- 5중 해자 → 4중 해자로 회귀
- IR 차별화 메시지 약화
- 자문 트랙 매출 추정 50% 하향 (M24 6,000만 → 3,000만)
- 광고주 풀 확장 효과 약화

### 완화책
1. **OB 모임 직접 영업**: 삼성 OB·LG OB·SK OB·현대 OB·롯데 OB·포스코 OB 6개 OB 모임 핵심 인사 직접 컨택. 핵심 1~2명만 모셔도 입소문 효과.
2. **한국전직지원협회 MOU**: 협회 회원사 통한 일괄 가입 채널 확보.
3. **창업진흥원·D.CAMP 파트너십**: 청년 스타트업 멘토링 프로그램 협업으로 욜드 멘토 자연 유입.
4. **시간당 단가 차별 검토**: 시간당 5만원 (대기업 컨설턴트 1/10) → 일감 자체 매출과 무관하지만 욜드 시장 진입 인센티브.
5. **욜드 큐레이션 매니저 채용**: 대기업 인사·영업 출신 1명, OB 인맥 활용 직접 모집.

## 통합 리스크 관리 원칙

1. **월간 리스크 리뷰**: 김연재 + CPO + 전략팀장 월 1회 리스크 점검 회의, docs/meetings/에 기록
2. **임계값 기반 트리거**: 각 리스크에 임계값 설정(예: M9 시점 MAU < 1만이면 R1 활성). 임계값 초과 시 즉시 완화책 실행.
3. **누적 리스크 지수**: 5개 리스크의 가중 평균을 매월 0~100 점수화, 60점 초과 시 전략 재검토 회의.
4. **투명한 투자자 커뮤니케이션**: VC 미팅에 본 리스크 평가서 첨부, 보수 시나리오 수치 인용.
