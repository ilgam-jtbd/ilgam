## 1. 시장 개요

### 글로벌 긱이코노미 성장

| 지표 | 2024 | 2032 (전망) |
|---|---|---|
| 플랫폼 기반 긱이코노미 시장 규모 | 5,567억 달러 | 1조 8,470억 달러 |
| 연평균 성장률 | — | ~16% CAGR |

### 한국 스폿워크 시장

- **초단기 근로자(주 15시간 미만)**: 2023년 기준 182만 명 (사상 최고, 2020년 이후 지속 증가)
- **저출산 구조적 배경**: 2024년 합계출산율 0.75 (OECD 최저), 생산가능인구(15~64세) 감소
- **주요 수요 업종**: 물류센터, 외식(서빙·주방보조), 편의점·마트, 병원·요양, 농업·수확

### 일본 스폿워크 시장 (Timee 기준)

- 2022년 648억 엔 → 2023년 약 800억 엔 (연 30%+ 성장)
- Timee: 누적 1,100만 다운로드, 메르카리 하로(3개월 500만 가입자 달성) 추격
- 이용자 34%가 "원하는 시간에 일할 수 있다"를 선택 이유로 꼽음
- 이용자 1/3 이상이 정규직 직장인의 부수입 목적 → **한국 시니어 세대와 유사한 동기**

### 2-1. Timee (タイミー, 일본) — 레퍼런스 플랫폼

| 항목 | 내용 |
|---|---|
| 설립 | 2018년 |
| 비즈니스 모델 | 스킬바이트(Skill Byte): 이력서·면접 없이 즉시 지원 → 근무 후 즉시 정산 |
| 수수료 | 구인자 과금 (채용 성공 수수료 방식, 워커 수수료 없음) |
| 주요 업종 | 외식, 물류, 소매, 의료·복지, 이벤트 |
| 핵심 UX | ① 프로필 3분 작성 → ② 공고 탭 하나 → ③ 지원 1탭 → ④ 당일 근무 → ⑤ 즉시 입금 |
| 차별점 | 워커 평점·뱃지 시스템, 기업 담당자 앱 내 소통, 당일 정산 |
| 한국 연결 | 급구(Needer Inc.) 투자 — Timee 첫 해외 투자 (2024) |

**ILGAM에의 시사점:**
- 이력서·면접 없는 "즉시 지원" 플로우는 시니어층에 특히 강력 (기술적 장벽 최소화)
- 즉시 정산(당일/근무 직후)은 고령층의 현금 흐름 니즈와 정확히 부합
- 뱃지·평점 시스템은 장기 참여를 유도하는 gamification

### 2-3. 당근알바 (Karrot / 당근마켓)

| 항목 | 내용 |
|---|---|
| 기반 | 당근마켓 지역 네트워크 (3,600만 인증 사용자) |
| 핵심 UX | "걸어서 10분 거리" 하이퍼로컬 매칭, 번호 노출 없는 채팅 |
| 일자리 유형 | 배달, 청소, 주방보조, 캐셔, 아이돌봄, 요양보호, 바리스타, 운전자, 선생님 |
| 지원 방식 | 공고 → 채팅으로 약속 → 출근 (앱 내 정산 없음, 직거래) |
| 수수료 | 현재 무료 (지역 생태계 확장이 우선) |
| 시니어 친화성 | ★★☆ — 지역 기반 신뢰는 강점, 앱 UX는 시니어 최적화 아님 |

**ILGAM에의 시사점:**
- 하이퍼로컬(동네) 매칭은 시니어가 출퇴근 거리 부담 없이 참여 가능
- 당근알바의 무료 정책으로 소규모 구인자(식당·편의점)가 익숙해진 상황 → ILGAM은 정산 자동화·평점·보험 연계로 차별화
- 직거래 방식은 임금 미지급 분쟁 위험 → ILGAM 에스크로 결제가 명확한 강점

## 5. ILGAM 포지셔닝 전략

### 핵심 차별화 (Blue Ocean)

1. **시니어 UX 최우선** — WCAG AAA (7:1 대비), 18pt 기본 글꼴, 48dp 터치 타겟
   - Timee, 급구, 당근알바 모두 일반 사용자 대상 설계 → 시니어 특화 공백
2. **에스크로 보호 결제** — 워커 임금 미지급 원천 차단 (당근알바 직거래 대비 명확한 우위)
3. **하이퍼로컬 + 자격증 매칭** — 동 단위 지역 필터 + cert_codes (지게차·요양보호사) GIN 인덱스
4. **이중 신뢰 시스템** — 구인자 승인제 (admin-web) + 워커 평점·멘토 뱃지

### 단기 Product Roadmap 연계

| 이슈 | ILGAM 기능 | 경쟁사 갭 |
|---|---|---|
| [B-1] match-engine | 시급·거리·cert_codes 3축 매칭 | 당근알바 없음, 급구 기본 필터만 |
| [B-3] payment-settle | PortOne 에스크로 자동 정산 | 당근알바 직거래, 급구 수동 정산 |

| [F-2] AI 추천 | cert_codes 기반 개인화 추천 | Timee JP만 구현, 한국 공백 |

## 6. worker-app 일감 카드 UX 설계 원칙

경쟁사 분석 결과 도출한 ILGAM 시니어 UX 요구사항:

### 카드 필수 정보 (정보 우선순위 순)
1. **공고명** — 18pt Bold (무슨 일인지 즉시 파악)
2. **시급** — 16pt, 강조색 (얼마 받는지 2순위)
3. **시간** — 14pt (언제 일하는지)
4. **거리** — 14pt (얼마나 가까운지)
5. **카테고리 뱃지** — 12pt (어떤 종류인지)
6. **즉시정산 뱃지** — 신뢰 지표 (Timee 학습)

### 접근성 기준 (non-negotiable)
- 터치 타겟 ≥ 48dp (한 손 조작 가능)
- 텍스트 대비 ≥ 7:1 WCAG AAA
- 200% 시스템 줌에서도 레이아웃 유지
- 카드 전체 터치 영역 (작은 버튼 지양)

### 카테고리 필터
- 물류·배송 / 외식·카페 / 청소·환경 / 유통·판매 / 돌봄·의료 / 농업·자연
- 수평 스크롤 탭 (6개 → 스크롤 필요 없는 3~4개 우선 표시)
