# ADR-002 · North Star Metric · M1~M3 MVP 범위

- **상태**: Accepted
- **일자**: 2026-04-23
- **의사결정자**: 김연재(PO) · CPO
- **참여 에이전트**: CPO, PO

## North Star Metric

**월간 완료 매칭 수 (Matched Shifts Completed per Month)**

MAU·GMV가 아닌 "한 번의 일이 실제로 끝났다"는 사건만이 양면시장의 진짜 가치를 포착한다. 구인자 등록 → 워커 지원 → 수락 → 출근 → 정산 5단계 퍼널을 모두 통과한 유일한 지표이며, 수수료 기반 수익모델과 선형 연동된다. 타이미도 상장 전까지 이 지표를 핵심으로 추적했다.

## M1~M3 필수 기능 5개 (우선순위 순)

1. **원탭 지원** — 지원서·자기소개 없이 "이 일 할래요" 한 번으로 종결
2. **카카오 알림톡 기반 매칭 수락·출근 리마인더 자동화**
3. **오프라인 본인확인 가능한 동네지사 연동 온보딩** (SMS 인증 실패 시 폴백)
4. **구인자 측 반복 공고 1클릭 재게시**
5. **당일 또는 D+1 자동정산** (PortOne 에스크로)

## 의도적 배제 3개

- **인앱 채팅**: 채널톡 단일화, 워커↔구인자 직통 금지
- **별점·후기 시스템**: 초기 표본 부족 시 부정 편향 폭주
- **AI 추천 피드**: Claude API는 백오피스 공고 정제에만, 워커 UX는 시간·거리 결정형

## 양면시장 포화 순서

**워커 선(先) 포화**. 60~74세 디지털 온보딩 이탈 60~70%(정보화진흥원) 실증 기준, 공급 곡선이 비탄력적. 구인자 오면 매칭 풀 없어 즉시 이탈.
- M1: 서울 3개 구(강서·송파·성북) 워커 1,200~1,500명(구당 400~500명) 오프라인+공공 MOU 선확보
- 구인자 영업은 "이미 대기 중인 시니어 풀" 카드로 B2B 직접 접촉

## 버티컬 선택

- **남기는 2개**: 물류·유통(쿠팡·IKEA 급구 검증 + 시니어 피킹/검수 적합) · F&B(오전 프렙·홀서빙 교체 + 대면 친화력 프리미엄)
- **버리는 3개**: 제조현장(산재보험 구조·안전교육 비용 불감당), 관광레저(계절성·CAC 붕괴), 교육멘토링(반복 빈도 주 1회 미만 → 스팟 정의 이탈)
- 사무보조는 M4 이후 보류 풀

## 지리 · 채널 · 첫 앵커

- **지리**: 서울 강서·송파·성북 3개 구 (급구 부산 본사·영남 강세 회피, 수도권 선점)
- **워커 월 833명 획득 채널 배분** (블렌디드 CAC 12,000원 목표):
  - 공공 MOU 45% (시니어복지관·50플러스재단·구청 일자리센터, CAC 8~12천원)
  - 알림톡 + 지인추천 30% (2천원 인센티브 + 알림톡 15원, CAC 7~9천원)
  - 지역 커뮤니티·교회·경로당 15% (CAC 15천원, 전환율 高)
  - 디지털 광고 10% (카카오·네이버, CAC 25~35천원)
- **첫 앵커 구인자 선정 기준**: ① 파트너 단가 4만원/회 이상(수수료 17.5% 시 7천원 확보) ② 월 반복채용 200회 이상 구조적 교대 결원 ③ 피어 그룹 레퍼런스 확산성 — 수도권 중견 3PL 또는 베이커리·죽 프랜차이즈 본사 우선

## 수수료 정책

- **M1 시작**: 구인자 15%, 워커 0%
- **계단식 인상**: 월 완료 매칭 3,000건 돌파 시 구인자 17%, 5,000건 돌파 시 20%
- 워커 수수료는 진입 창문(9~12개월) 내내 0% 유지 — "돈 떼가는 앱" 브랜딩 방지가 비가역 자산

## 집착 지표

**D7 리텐션 30%**. D1은 온보딩 UX 허수, D30은 월 1회로도 유지돼 매출 상관 약함. D7은 "첫 주 내 두 번째 지원" 포착 → 매출 곡선과 선형. 보조로 D30 20% 모니터.

## 시니어 UX 보스 판단 기준 3개

1. **되돌릴 수 없는 액션 수** — 종단 액션 24시간 내 1탭 롤백 기본값
2. **다음 행동이 불명확한 화면 수 = 0** — 모든 상태는 단일 다음 행동 명시
3. **전화하고 싶은 순간 수** — 0 수렴 목표. 미수렴 시 채널톡·알림톡·지사 중 어느 축 파손인지 역추적 계측

## 반직관적 결정

**"시니어 앱에 검색창 없음"**. 베이비부머 2차는 검색 타이핑이 인지부하 피크. 위치·요일·업종 3종 태그 필터 + 동네지사 오프라인 직원 대면 상담으로 검색을 인간에게 위임. 당근·급구가 따라올 수 없는 해자.

## 6개월 실패 신호 3개

- M3 월 완료 매칭 < 1,500건 (양면시장 점화 실패)
- 워커 D30 < 20% (3중 UX 미작동 → 동네지사 밀도 재설계)
- 채널톡 문의 중 "전화 걸 수 있나요" 비중 주 15% 초과 3주 연속 (CX 정책 구조적 어긋남 → 지사 유선 우회번호 중간 타협 테스트)

2개 이상 동시 발생 시 김연재(PO)에 공식 피벗 의제 상정.
