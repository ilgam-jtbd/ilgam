## 핵심 피벗 요약 (v2.2)

**인력사무소 디지털화 + 시니어 단독 점유 = Blue Ocean.** 5중 해자 결합:
1. **단계화 수수료** (Phase 0 양측 0% → M6 법인설립 + 10% 인력사무소 표준)
2. **시니어 UX** (ADR-003 기반)
3. **공공 MOU** (한국노인인력개발원·50플러스재단)
4. **컨텐츠 QA** (ADR-010 4-Tier)
5. **YOLD 자문 트랙** (50~64세 대기업·중견·스타트업 출신 멘토링)

v2(2026-04-24)는 "양측 0% Phase 1 M0~M18 + 광고 Phase 2 M18+"였으나, **v2.2(2026-04-28) 결정으로 Phase 0 (M0~M5 무료 검증) → M6 법인설립 + 10% 수수료 도입(인력사무소 오프라인 표준)** 으로 단축. 광고·자문 멤버십은 M18~M30 인플렉션 단계에서 부가.

근거 보강:
- 국내 인력사무소 ~3만곳 · 10% 수수료 표준 → 디지털 전환 공백 = 단독 점유 가능 시장
- 5개월 무료 검증으로 신뢰 자산 + 데이터 확보, M6 법인 설립과 동시 10% 도입 시 "인력사무소 표준 채택" 프레임이 시니어·구인자 양쪽 거부감 최소화
- Seed ₩3억 + 정부과제 ₩5천만 → TIPS ₩10억 → Pre-A ₩10~15억 자금 트랙 정합성

## North Star Metric (3-factor)

**월 활성 검증 공고 수 (MAVJ) × MAU × 신뢰 지수**

```
NSM = MonthlyActiveVerifiedJobs × MAU × TrustScore
```

- **MAVJ**: `qa_status='approved'` 상태 + 한 건 이상 지원받은 공고. 단순 게시물이 아닌 "실제로 매칭 시도가 일어난 검증된 공고".
- **MAU**: 30일 이내 1회 이상 앱·웹 접속 워커. 중복·봇 제외.
- **TrustScore (0~1)**: `1 - 신고건수 ÷ 검증공고수 - 자동반려율 보정`. M3 베이스라인 0.7+ 목표.

이 NSM은 (a) 무료 모델에서도 가치 측정 가능, (b) QA 시스템 작동 여부와 직접 연동, (c) 광고 모델 도래 시 광고 인벤토리 가격(검증된 공고 × MAU)으로 자연스럽게 환산.

## M1~M3 필수 기능 5개 (재정의)

1. **원탭 지원** — 지원서·자기소개 없이 "이 일 할래요" 한 번으로 종결
2. **카카오 알림톡 자동화** — 매칭 수락·출근 리마인더·QA 결과 통보
3. **동네지사 연동 온보딩** — SMS 인증 실패 시 폴백
4. **구인자 1클릭 재게시** — QA pre-pass된 이전 공고는 자동 approved 빠른 노출
5. **컨텐츠 QA 큐 (운영자)** — 신규 공고 자동 분류 + 운영자 검토 + 신고 처리. ADR-010 상세.

## 의도적 배제 4건

- **수수료 시스템**: Phase 1(M0~M18) 양측 0%. 코드에서 `payments` 흐름 폐기, PortOne 광고 결제용으로만 보존.
- **매칭 자동 LLM**: 시급/거리/시간 결정형 유지. Claude API는 컨텐츠 QA + CX 트리아지에만.
- **인앱 채팅**: 채널톡 단일화, 워커↔구인자 직통 금지.
- **별점·후기**: 초기 표본 부족 시 부정 편향 폭주.

## 양면시장 포화 순서

**워커 선(先) 포화** (이전과 동일).
- M1: 서울 3개 구(강서·송파·성북) 워커 1,200~1,500명 오프라인+공공 MOU 선확보.
- 구인자 영업: **"무료 + 검증된 시니어 풀"** 카드. 수수료 0% + QA 통과 공고만 노출 = 알바천국·당근알바 대비 단가 추가 부담 없이 신뢰성 ↑.

## 워커 세그먼트 (3 트랙)

| 트랙 | 연령 | 출신 | 주 매칭 카테고리 | 시급대 |
|---|---|---|---|---|
| **A. 베이비부머 2차** | 1964~1974생 (52~62세) | 일반 사무·생산·서비스 경력 | 물류·F&B·청소·유통 | ₩10,030~15,000 |
| **B. 70대 시니어** | 1955~1964생 (62~71세) | 사무·서비스·생활기술 | F&B·돌봄·청소·농업 | ₩10,030~13,000 |
| **C. 욜드족 (YOLD, 신규)** | 65~75세 | **대기업 임원·CEO·전문가** | **자문·멘토링 (consulting)** | **₩50,000~300,000** |

**욜드족(Young Old) 추가 사유** (2026-04-28):
- 한국 65세 이상 1,000만 명 돌파(2024) 중 약 60~70%가 욜드족 (경제력·학력·디지털 친숙)
- 대기업 임원·CEO 출신, 스타트업·중견기업 자문 경험 보유
- 기존 알바 플랫폼이 "단순 시급 노동"으로만 시니어를 한정 → ILGAM은 **"경험 자산 매칭"** 트랙 추가
- B2B Premium 광고 슬롯(ADR-011 Phase 2)의 핵심 콘텐츠 풀이 됨 — 중견기업이 자문 의뢰 시 검증된 욜드족 풀에 우선 매칭

**자문(consulting) 카테고리 신규**:
- `JobCategory` enum에 `consulting` 추가 (packages/core)
- 시급 정상 범위 ₩50,000~300,000 (Tier 1 시급 외삽 룰 면제, ADR-010)
- 매칭 패턴: 시간제(2~8시간/주) · 프로젝트제 (1~3개월) · 정기 자문(월 1~4회)
- 대상 구인자: 시드~시리즈 B 스타트업, 매출 50~500억 중견기업, 정부 R&D 과제 멘토링

## 버티컬 선택

- **유지 2개**: 물류·유통, F&B (이전 결정 유지)
- **신규 추가** (욜드족 세그먼트와 함께): **자문·멘토링 (consulting)** — 스타트업·중견기업 자문, 정부 R&D 과제 멘토링
- **버리는 3개**: 제조현장, 관광레저, **(교육) 단순 강사** (자문은 별도 카테고리로 분리)

## 지리 · 채널 (유지 — 무료 모델로 CAC 합리성 재확인 필요)

- **지리**: 서울 강서·송파·성북 3개 구 (수도권 선점)
- **워커 월 833명 채널 배분**: 공공 MOU 45% / 알림톡+지인 30% / 지역 커뮤니티 15% / 디지털 광고 10%
- **CAC 12,000원 목표 유지**. 단 수수료 매출 0이므로 CAC 회수가 광고 매출(M18+)에 의존 → 자본 소요량 1.5~2배 재산정 필요(전략팀장 별도 계산).

## 수수료 정책 (Phase 구분 · v2.2 단계화)

### Phase 0 — 무료 검증 (M0 ~ M5, 2026.10 ~ 2027.02)
- **워커·구인자 모두 0%** — 5개월 무료 검증
- 직거래 모델 (구인자 → 워커 직접 송금). 플랫폼 미개입.
- 목표: 워커 5,000 + 검증 공고 5,000건 + 데이터·신뢰 자산 축적
- BEP 무관, 자본 소진 속도만 모니터

### Phase 1 — 법인 + 10% 수수료 (M6+, 2027.04~)
- **법인 설립** 동시 **구인자 10% 수수료 도입 (인력사무소 오프라인 표준)**
- **워커는 영구 0%** ("돈 떼는 곳" 인식 차단 = 비가역 자산)
- 정산 흐름 부활: PortOne 에스크로 → 워커 송금 (ADR-004 v2.1 §정산 모델)
- "인력사무소 표준 채택" 프레임 — 구인자 거부감 최소화 (디지털 vs 오프라인 동일 단가)
- M6 트리거: 워커 5K+ + 검증 공고 5K+ + 법인 설립 완료

### Phase 2 — 광고 + 자문 멤버십 인플렉션 (M18~M30)
- 메인 매출은 여전히 **10% 수수료**, 부가:
  - 상단 노출 CPC ₩250 (워커 앱)
  - B2B Premium 멤버십 ₩50~200만/월 (구인자 우선 매칭 + 분석)
  - **자문 매칭 멤버십 ₩50~200만/월** (욜드 자문 트랙, 스타트업·중견기업 구독)
- M18 트리거: MAU 15만+, NPS +30+, 운영자 QA 처리량 안정
- BEP M22 (월 ₩3억). 이전 v2 "M18 광고 BEP" 가정에서 v2.2는 수수료 매출 우선 반영으로 BEP 4개월 단축.
- 코드는 architecturally ready (ADR-011 v1.1, dormant 구조 + M18 토글)

## 집착 지표 (재정의)

- **D7 리텐션 30%** (유지)
- **신뢰 지수 0.7+** — 신고/검증 비율, 자동반려율 보정
- **운영자 QA 처리량 분당 5~10건** (지속 가능성 지표)

## 실패 신호 (Phase 0 종료 M5 + 6개월 평가 M11)

### M5 게이트 (Phase 0 → Phase 1 법인 설립 결정)
- 워커 < 5,000명 OR 검증 공고 < 5,000건 → **법인 설립 1~3개월 연기**
- 신뢰 지수 < 0.6 → **QA 시스템 보강 후 M7 재평가**
- KORDI MOU 본 체결 미진행 → **공공 채널 백업(50플러스재단 단독)** 으로 자본 1.5배 재산정

### M11 평가 (Phase 1 안정화 후 6개월)
- 월 검증 공고 < 월 매출 ₩600만 (10% × GMV ₩6,000만) → **수수료율 12% 검토 또는 워커 5,000명 추가 락인**
- 워커 D30 < 20% (3중 UX 미작동 → 동네지사 밀도 재설계)
- QA 자동반려율 > 25% OR 수동 검토 큐 적체 > 24h
- 채널톡 문의 중 "전화 걸 수 있나요" 비중 주 15% 초과 3주 연속

2개 이상 동시 발생 시 김연재(PO)에 공식 피벗 의제 상정.

## 시니어 UX 보스 판단 기준 3개 (유지)

1. 되돌릴 수 없는 액션 수 — 24h 1탭 롤백 기본값
2. 다음 행동이 불명확한 화면 수 = 0
3. 전화하고 싶은 순간 수 — 0 수렴

## 반직관적 결정 (유지)

**"시니어 앱에 검색창 없음"**. 위치·요일·업종 3종 태그 + 동네지사 직원 대면 상담.
